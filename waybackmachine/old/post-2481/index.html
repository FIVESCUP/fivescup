新体制のお知らせ

このたび”FIVESCUP”はLive on Threeから離脱・独立したことをお知らせいたします。
元々 FIVESCUP はコミュニティ「SOLIDUS.FIVE」が運営を行っていましたが、元組織のMohimohi Server との統廃合が行われた後、運営チームが「Live on Three」へと合流し、これまで運営を行って来ました。
ですが、合流した事による開催頻度の低下などの事情により再度独立し、活動を再開することと致しました。

それに伴い、以下のリンクが変更となります。

  • Twitter : @SOLIDUSFIVE → @FIVESCUP
  • twitch.tv/lo3_jp (Live on Threeチャンネル) → FIVESCUP
  • https://fivescup.lo3.jp → https://fivescup.jp

    また、今後のイベントにはこれまで採用していたLive on Threeルールを廃止し、FIVESCUPルールに準拠した運営・対応を行います。
    これからもFIVESCUPを宜しくお願い致します。

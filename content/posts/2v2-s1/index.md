4月21日(土)に、**FIVESCUP 2v2 RetakeMasters: Season1** を開催します。

	**FIVESCUP 2v2 RetakeMasters** は、爆弾設置状態から開始する「リテイクMOD」を使用し、CS:GOの醍醐味の1つである少人数戦の連携を気軽に楽しむことをコンセプトとした、2v2の大会です。

		
## FIVESCUP 2v2 RetakeMasters: Season1

| 項目 | 内容 |
| --- | --- |
| 大会名 | FIVESCUP 2v2 Retake: Season1 |
| 対戦形式 | グループステージ + プレーオフ |
| フォーマット | Best of 1 (MAP: Inferno) |
| 開催日程 | 2018年 4月 21日 (土) |
| 最大チーム数 | 32 チーム |
| ルール | [Live on Three 大会共通ルール](../../rules/) |

### 賞品

	優勝: 1,000円相当の武器スキン

### 参加登録方法

	Battlefyにて行います。

	本大会から、チームで設定するインゲームネームをBattlefyのユーザーネームと統一し、参加時のフォームに **SteamID64** を入力するようにルールが変更となっていますのでご注意ください。

(埋め込み)

### 大会進行

	[Live on Three Discord](https://discord.gg/x8cNueE) にて行います。

#### スケジュール

スケジュールは予期せず変更される場合があります。

- 17:45 … チェックイン開始

- 18:45 … チェックイン締切

- 18:50 … グループ発表

- 19:00 … グループステージ 第1試合

- 19:30 … グループステージ 第2試合

- 20:00 … グループステージ 第3試合

- 20:30 … プレーオフ 準々決勝

- 21:00 … プレーオフ 準決勝

- 21:30 … プレーオフ 決勝

### ストリーミング配信

	[twitch.tv/FIVESCUP](https://www.twitch.tv/fivescup)

### FIVESCUPとは？

	**FIVESCUP** は、[Live on Three](https://lo3.jp) が主催する **Counter-Strike: Global Offensive** のオンライン大会です。国内競技シーン発展のため、誰でも気軽にCS:GOの競技的側面を楽しんでいただけるような大会・イベントを企画運営しています。

2020年5月16日よりの31日までの間、**FIVESCUP RUSHERS**を開催します。

![  ]

**FIVESCUP RUSHERS**は、毎週日曜日に行われるCS:GOの5v5トーナメントです。

競技マップを少数に絞り、その習熟度を競う事をコンセプトに開催されます。

		
## FIVESCUP RUSHERS 2020

| 大会名 |
| FIVESCUP RUSHERS: 2020 Mar |

| 対戦形式 |
| トーナメント戦 |

| フォーマット |
| Best of 1. |

| MAP POOL |
| 毎週7mapよりランダムに選出いたします。 |

| 開催日程 |
| 2020年 05月 毎週日曜日(24日を除く) |

| 最大チーム数 |
| 各日 16 チーム |

| ルール |
| [大会共通ルール](../../) |

### 参加登録方法

公式Discord Project GIARSより行います。

Project GIARSの使い方は **[こちら](../giars-howto/)**

### 大会進行

**[FIVESCUP Discord](https://discord.gg/Uh7qQcp)** にて行います。

### ストリーミング配信

Project GIARSの自動配信システムを使ったクリーンフィード配信を行います。

(埋め込み)s5projectsのライブビデオをwww.twitch.tvから視聴する

### FIVESCUPとは？

**FIVESCUP** はS5Projectsが運営・主催する **Counter-Strike: Global Offensive** のオンライン大会です。コミュニティを重視した大会を目指しています。

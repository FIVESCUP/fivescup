2025年2月24日に、**FIVESCUP EXPERIMENTAL SEASON2**を開催いたします。

**FIVESCUP EXPERIMENTAL**は、FIVESCUPブランドの元行われる5v5の大会型イベントです。  
特殊なルールと環境を用意し、新規技術の実証実験、及び技術研究を目的として開催しています。

## FIVESCUP EXPERIMENTAL: SEASON2

| イベント名 | FIVESCUP EXPERIMENTAL: SEASON2 |
| --- | --- |
| 対戦形式 | シングルエリミネーショントーナメント |
| フォーマット | Best of 1. / オンライン |
| SEASON2 MAP POOL | Inferno / Vertigo / Train |
| 開催日程 | 2025年 02月24日 |
| 最大チーム数 | 8チーム |
| ルール | [大会共通ルール](../../rules/) |
| 開催目的 | DiscordBOTを使用したイベント進行の半自動化、DEMOの自動公開 |

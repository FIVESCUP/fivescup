12月16日(日)に、**FIVESCUP Challenge: Season3** を開催します。

**FIVESCUP Challenge** は、7つある競技マップを1つに絞り、そのマップの習熟度を競うことをコンセプトとした、5v5の大会です。

		
## FIVESCUP Challenge: Season3

| 大会名 |
| FIVESCUP Challenge: Season3 |

| 対戦形式 |
| グループステージ + 決勝戦 |

| フォーマット |
| Best of 1 (MAP: Mirage) |

| 開催日程 |
| 2018年 12月 16日 (日) |

| 最大チーム数 |
| 8 チーム |

| ルール |
| [大会共通ルール](../../) |

### 賞品

優勝: 鍵5個

### 参加登録方法

[BATTLEFY](https://battlefy.com/fivescup/fivescup-challenge-season3/5bffa985fbd73003dd43337b/info)にて行います。

前回より、チームで設定するインゲームネームをBattlefyのユーザーネームと統一し、参加時のフォームに **SteamID64** を入力するようにルールが変更となっていますのでご注意ください。

SteamID64 の確認方法については [こちら](../steamid/) を参照ください。

[BATTLEFY](https://battlefy.com/fivescup/fivescup-challenge-season3/5bffa985fbd73003dd43337b/info)

### 大会進行

[Discord](https://discord.gg/Uh7qQcp) にて行います。

#### スケジュール

スケジュールは予期せず変更される場合があります。

- 17:45 … チェックイン開始

- 18:45 … チェックイン締切

- 18:50 … グループ発表

- 19:00 … グループステージ 第1試合

- 20:10 … グループステージ 第2試合

- 21:10 … グループステージ 第3試合

- 22:30 … 決勝戦

### FIVESCUPとは？

**FIVESCUP** はFlow が運営・主催する **Counter-Strike: Global Offensive** のオンライン大会です。コミュニティを重視した大会を目指しています。

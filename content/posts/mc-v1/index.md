4月28日(土)の FIVESCUP Challenge: Season2 開催に合わせ、[CSGO Moviemake Community Japan](https://discord.gg/NtZpcSU) , [RapidWorks](https://rapidworks.jp) , [Azuleat氏](https://azuleat.com) と共同で **FIVESCUP Movie Contest Vol.1**を開催いたします。

		
## FIVESCUP Movie Contest: Vol.1

| イベント名 |
| FIVESCUP Movie Contest: Vol.1 |

| 概要 |
| FIVESCUP Challenge: Season2 の試合demoを用いた動画コンテスト |

| 開催日程 |
| 2018年 4月 28日 (土) ~ 5月 28日(月) |

### 賞品

- 最優秀賞 … 2,000円相当の武器スキン + サイトにて紹介

- 優秀賞x3 … サイトにて紹介

- RapidWorks賞([Azuleat氏](https://azuleat.com)選出) … サイトにて紹介

### 製作する動画について

	以下の条件を満たしている動画のみ、応募することが出来ます。

- FIVESCUP Challenge: Season2 の試合のdemoを使用している

- 動画時間は5分以内

- プレイヤー名やチーム名を隠していない

- プレイヤーを侮辱する内容ではない

#### 注意事項

- 実況解説の音声は必須ではありません。

	

- FIVESCUP Challenge: Season2 の試合であれば配信試合でなくても構いません。

- 大会に参加したプレイヤーも参加することが出来ます。

- 動画の収益登録や、動画内PRなどは自由です。

- 投稿された作品・作者の TwitterID は FIVESCUP の・SNSなどで公開されます。

### 提出方法

	動画URL提出を以って参加登録とします。

	5/28までに、[CSGO Moviemake Community Japan](https://discord.gg/NtZpcSU) の #fivescup-contest-vol1 チャンネルに、**Youtubeの動画URL** と **TwitterID** を提出して下さい。

### 素材

	[FIVESCUP Challenge Season2 demo](http://demos.lo3.live/demos/event/FIVES_CUP_CHALLENGE_Season2) [キャスター音声付動画](https://youtu.be/RwyOP4CNqbU) [大会ロゴ](https://drive.google.com/open?id=1bNfOoJKHyjXxzCgAAIF2xYwyRcz0jzZy)

### FIVESCUPとは？

	**FIVESCUP** は、[Live on Three](https://lo3.jp) が主催する **Counter-Strike: Global Offensive** のオンライン大会です。国内競技シーン発展のため、誰でも気軽にCS:GOの競技的側面を楽しんでいただけるような大会・イベントを企画運営しています。

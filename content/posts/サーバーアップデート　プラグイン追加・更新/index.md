SOVサーバーをアップデートしました。

		
## 対象サーバー:レンタルマッチサーバー

[SourceBans](http://solidusfive.xyz:8080/SourceBans/index.php?p=home)   を導入しました。
![  ]

本プラグインにより、レンタルマッチサーバーへ追加された全てのBAN対象IDは同期され、

BANされたプレイヤーは全てのレンタルマッチサーバーにアクセス不能になります。

BANされたプレイヤーリストはホームページ右上の「SourceBANS」より確認可能です。

 

## lo3 loader by MelonSodaをアップデートしました。

以下MelonSoda様より引用となります。

このアップデートは以下のサーバが対象です。

[JP] MelonSoda CS:GO Server
[LAN] MelonSoda CS:GO Server
[JP]Solidus-FIVE Rental Server #1 by ari

[JP]Solidus-FIVE Rental Server #2 by ari

[JP]Solidus-FIVE Rental Server #3 by ari

[JP]Solidus-FiVE CS:GO Event Server #1

by ari [JP]Solidus-FiVE CS:GO Event Server #2 by ari

lo3loader by MelonSodaを日本語化した。

![  ]

CS:GOのLanguageが日本語である場合は日本語、その他の言語である場合は英語で表示される。

録画されたDEMOはLanguageに関わらず英語で表示される。

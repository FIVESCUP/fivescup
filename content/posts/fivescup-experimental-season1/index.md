2024年6月30日に、**FIVESCUP EXPERIMENTAL SEASON1**を開催します。

**FIVESCUP EXPERIMENTAL**は、新規技術の実証実験、及び技術研究を目的とし、特殊なルールと環境下で実施される5v5の大会です。

## FIVESCUP EXPERIMENTAL: SEASON1

| 大会名 | FIVESCUP EXPERIMENTAL: SEASON1 |
| --- | --- |
| 対戦形式 | シングルエリミネーショントーナメント |
| フォーマット | Best of 1. |
| SEASON1 MAP POOL | Mirage / Inferno / Dust2 |
| 開催日程 | 2024年 06月30日(日) |
| 最大チーム数 | 8チーム |
| ルール | [大会共通ルール](../../rules/) |
| イベントのゴール | Discordコマンドを使用したトーナメントの作成、チームの作成・参加、試合の作成とサーバー設定の自動化・進行の自動化。 |

### FIVESCUPとは？

**FIVESCUP** はS5 Worksが運営・主催する **Counter-Strike** シリーズ のオンライン大会です。コミュニティを重視した大会を目指しています。
